StopError

StopError: StopError
  • Defined in src/core/promise.ts:210

Stopping The library also supports stopping the retry loop before the timeout occurs by throwing a new instance of retry.StopError from within the called function. The StopError constructor accepts one argument. If it is invoked with an instance of Error, then the promise is rejected with that error argument. Otherwise the promise is rejected with the StopError itself.
