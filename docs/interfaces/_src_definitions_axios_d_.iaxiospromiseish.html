Interface IAxiosPromiseish<R>

a NOT BLUEBIRD COMPATABLE, promise-like construct returned by axios calls. should NOT passs this through the system. you need to wrap it in a "new Promise((resolve,reject)=>{})" object. this interface actually is more promise-compatabile than what we expose, but we limit it to the minimum to force you to wrap in a 'new Promise()', otherwise BAD THINGS WILL HAPPEN TO YOU!
